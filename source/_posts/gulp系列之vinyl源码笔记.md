## vinyl简介
vinyl是gulp团队维护的一个虚拟文件格式库。Vinyl对象，主要包括path和contents属性，是一个描述文件的元数据对象。

<!-- more -->

## vinyl用法
```js
var Vinyl = require('vinyl');

var jsFile = new Vinyl({
  cwd: '/', 工作目录
  base: '/test/', base路径
  path: '/test/file.js', //
  contents: new Buffer('var x = 123')
});
```

通过Object.defineProperty定义的属性
- contents 文件内容
- cwd 工作目录
- base 基准路径
- relative 相对路径
- dirname 文件夹名称
- basename 文件名称（包括扩展名）
- stem 文件名
- extname 扩展名
- path 完整路径
- symlink 软链
